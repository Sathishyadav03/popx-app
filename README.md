# PopX React Assignment

A pixel-perfect React JS implementation of the PopX mobile UI design using React + Vite.

## Live Demo

https://popx-app-silk.vercel.app/

## GitHub Repository

https://github.com/Sathishyadav03/popx-app

---

# Features

- Pixel-perfect mobile-centered UI
- React Router navigation
- Register page
- Login page
- Account settings page
- Dynamic user data using localStorage
- Responsive design
- Clean component-based architecture
- Hosted on Vercel

---

# Tech Stack

- React JS
- Vite
- React Router DOM
- CSS3
